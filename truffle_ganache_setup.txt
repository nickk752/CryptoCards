TRUFFLE INSTALLATION 
nodeJS needs to be installed first

$ npm  install -g truffle

CREATING A PROJECT
$ mkdir myproject
$ cd myproject
$ truffle init

folder is now initialized with project structure
Contents include:
conracts/
migrations/
test/
truffle.js
